## 課題

### 課題1: 挨拶

コマンドライン引数として名前を受け取り、挨拶を表示するスクリプトを作ってください。
ただし、引数が指定されなかった場合は、"Hello, World!" と表示することとします。
また、シェルスクリプト本体のファイル名は `greet.sh` としてください。

```bash
# 実行例
$ ./greet.sh Alice
Hello, Alice!
$ ./greet.sh
Hello, World!
```

### 課題2: ファイル・ディレクトリ構造の作成

以下のようなファイル・ディレクトリ構造を作成するシェルスクリプトを作ってください。
提出においては、スクリプト本体のファイル名は `create_structure.sh` としてください。
また、実行結果として、作成されたディレクトリ構造をリポジトリに`push`してください。

```shell
project/
├── data/
│   ├── input0001.dat
│   ├── input0002.dat
│   ├── ...
│   ├── input0098.dat
│   └── input0099.dat
├── scripts/
│   └── process.sh
└── results/
    └── output.dat
```

### 課題3: 空白とワイルドカード

適当な空ディレクトリで、以下のコマンドを実行してください。

```bash
$ touch "data 01.txt" "data 02.txt" result.txt
```

その後、次の2つの変数を定義します。

```bash
filename="data 01.txt"
pattern="*.txt"
```

以下のコマンドを実行し、それぞれの出力を確認してください。

```bash
printf '<%s>\n' $filename
printf '<%s>\n' "$filename"
printf '<%s>\n' $pattern
printf '<%s>\n' "$pattern"
```

各コマンドを実行した場合の出力結果と、引用符の有無によって結果が異なる理由を、空白による分割とワイルドカードの展開について注意して説明してください。

実行結果及び説明を `assignment.md` に記述してください。

### 課題4: 自作コマンド (希望者のみ)

シェルスクリプトを用いて、自由にコマンドを作ってください。
ただし、作成するコマンドでは、コマンドライン引数、条件分岐、ループ、配列のうち、少なくとも1つを使用してください。

作成したシェルスクリプトのファイル名とコマンドの目的や実行例を `assignment.md` に記述してください。
シェルスクリプト自体も提出してください。
